== Installation ==

Download, Upgrading, Installation:

Upgrade

1. First deactivate Google Plus One Button
1. Remove the `google-plus-one-button` directory

**Install**

1. Unzip the `google-plus-one-button.zip` file. 
1. Upload the the `google-plus-one-button` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the Twitter Follow Button and a subpage for it (Google Plus One Button) will appear in your Options menu.

If you find any bugs or have any ideas, please mail me.
